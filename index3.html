VR書店

發想動機

　　VR可說是現代資訊科技的潮流之一，藉由裝置去構築出虛擬環境，讓人們可以突破現實的框架，實現許多在過往難以達成的應用。除了娛樂以外，在教育、醫療等各項領域，近年來也紛紛投入VR的應用開發。

　　科技發展帶來了嶄新的生活型態，相對的，過往的生活型態也可能因為方便性和即時性等現實因素，而面臨淘汰的絕境。傳統書店正是其中之一，現在無論是影片還是文章，大部分都能在網路上找到並完成購買與瀏覽，儘管還有人喜愛書店的平和氛圍，但實體書店在商場上的競爭力逐年下滑，已經是不爭的事實。

　　不過，透過VR的技術，或許我們有辦法在虛擬環境中，重現傳統書店的氛圍也說不定。

目標受眾&預期成效

  • 書店經營者→顧客

    　　這套系統可能沒辦法套用在現有的實體書店，不過我們期望，它能夠為書店的經營方式開創一條新的道路。

    　　透過VR沉浸式體驗的特色，將原本書店中顧客的各種行為模式，以及經營的環境，全部搬到虛擬環境中延續下去。不只如此，透過VR或許還可以強化這些體驗，讓使用者處在一個更加舒適的環境。

預想功能

需要增加或刪減的部分可以再討論。

  • 個人帳戶功能

    可以連結APP記下一些個人事項，也方便VR設備追蹤使用者

  • APP

    可以查詢到訪店家的時間、連結記事本與購物車、線上訂書等功能，並且還有電子書閱讀器，可以閱讀在裡面購買的書。

  • 虛擬空間設計

    目前構想是：可以像網咖一樣在一個人的包廂裡使用VR設備，裡面的虛擬環境則是設計成書籍區、商品區、餐飲區

    使用者
    書籍區
    商品區
    餐飲區

    ▲VR環境內的小地圖，由於希望使用者能坐著操作，因此空間不大

  • 聊天功能

    可以透過個人帳戶找到其他VR裝置的使用者，進行語音通話。

互動流程

  • 使用裝置

    抵達設備所在地(一個人且有地方坐的獨立空間)
    → 戴上裝置
    → 登入個人帳戶

  • 書籍區

    透過分類或關鍵字搜尋
    → 選擇書籍
    → 選擇購買、放入購物車或現場閱讀(在閱讀時，背景會隨著你選擇的書籍而變化)
    → 如果選擇購買，再選擇要現場領書、儲存電子書或等待郵寄

  • 商品區

    透過分類或關鍵字搜尋
    → 選擇商品
    → 顯示商品簡介
    → 選擇是否購買或放入購物車
    → 前往櫃檯領貨或等待郵寄

  • 餐飲區

    顯示菜單
    → 點餐
    → 等待餐點送到包廂

特色與優勢

  • 跨領域的科技結合

    將資訊科技套用在書店的經營方式上。

  • 新型態的書店經營模式

    VR的沉浸式體驗，以及簡化過的操作流程

  • 台灣還沒有類似案例

    國外雖然有類似的案例，但台灣目前還沒有查到以VR方式經營的書店
